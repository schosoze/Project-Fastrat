This script allows you to bend the object, causing it to take the form of a spline. This flexible and easy to use tool is a great help for level-designer: Now there is no need to adjust roads or fences in a 3d editor. With the help of this script you can create objects of any length and complexity just from a repeated straight segment. It allows you to save money on buying expensive specialized road tools, and has far more features - with SplineBend you can bend and tile and any type of extended objects: railways, pipes, wires, tunnels, caves, and many, much more!

Watch this video to see how it works.

Quick Tutorial

1. Put the object you are going to bent on a scene. When creating an object, consider that it will bend and tile in Z axis, and the top must llok in direction Y.

2. Add SplineBend component (Scripts -> SplineBend). On both sides of the object two primary markers will appear.

3. Place the markers to the desired trajectory. In order to move the marker - highlight it by clicking and drag it the same way as an ordinary game object. To add a marker - Ctrl-right click on the line between the markers. To delete a marker - just select it and press Delete key.

4. Set the selected marker type in the inspector:

  • smooth automatically creates a smooth curve between two adjacent markers;
  • transform sets the spline direction by transform rotation and smoothness by transform scale;
  • beizer allows to configure spline manually with handles.
  • beizer corner handles can lie not only on a straight line, allowing you to create sharp edges;
  • corner always creates sharp corners without smoothing.

    • 5. The tiles tab sets the desired tile count - the number of repetitions of this object in length.

    6. If you want to spread the mesh to the terrain or other mesh - for example, a road - in the tab Drop to Terrain enable Drop.

    Properties

    MarkersArray of markers used by current script instance.
    AxisRotates model to make it tile along chosen axis.
    Marker SizeSize of displayed marker gizmos.
    EqualizeEvently distributes mesh verts along spline. Helps to avoid density heterogeneity in beizer spline.
    MeshesObject meshes:
    Initial Render MeshSource of a graphical mesh. If object mesh is missing - assign it as Initial Mesh from file or prefab.
    Initial Collision MeshSource of a collision mesh. If object mesh is missing - assign it as Initial Mesh from file or prefab.
    Render MeshCurrent MeshFilter mesh.
    Collision MeshCurrent collider mesh.
    TilesRepeating mesh geometry settings:
    Tile CountNumber of times mesh repeated.
    Tile OffsetTile length. Distance between same points in different tiles.
    Reset Tile OffsetGets tile offset using initial mesh bounding box.
    Drop to TerrainPlaces mesh at the surface of terrain or other collision mesh:
    DropTurns Drop to Terrain on and off.
    Seek DistanceDrops vert to terrain only if distance to terrain is less than Seek Distance.
    Terrain LayerLayer number of terrain object.
    Height OffsetRaises or lowers vert after Drop to Terrain.
